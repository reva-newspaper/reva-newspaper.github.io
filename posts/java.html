PROGRAMMING IN JAVA INTERNAL EXPERT LECTURE REPORT


Date: 13​TH​ FEBRUARY 2019
Time: 10:30 AM to 11:50 PM
Venue: Room No.310, 3​rd​ Floor C V Raman Block.
Topic: Programming in Java internal expert lecture.

The School of EEE had organized an Expert lecture on Programming in Java on 13th February, 2019 between ​10:30 AM to 11:50 PM in ​C V Raman Block​for 6​th semester B.Tech Students. The event was coordinated by Dr. Vishwanath R Hulipalled, Professor School of Computing and Information Technology, Deputy Director R&I Council and focused on the importance of Object Oriented Programming and Java in the industry. Students learnt how to use some of the concepts of Object Oriented Programming in developing the software. He gave an insight into the various job opportunities as well as tips on interviews and the kind of questions that could be asked
